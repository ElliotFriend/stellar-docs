General node health check.

## Parameters

- None

## Returns

- `<object>`
  - `status`: "healthy"

## Examples

### Request

```json
{
  "jsonrpc": "2.0",
  "id": 8675309,
  "method": "getHealth"
}
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 8675309,
  "result": {
    "status": "healthy"
  }
}
```
